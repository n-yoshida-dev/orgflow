# ADR-010: 同一承認段に複数承認者がいる場合のルール

## Context

1つの承認段に複数の承認者が存在する場合、
承認完了の判定ルールを定義する必要がある。

## Decision

- Phase1では以下とする
  - 同一承認段に複数承認者がいる場合
  - 「誰か1人が承認すれば次の承認段へ進む」

- 全員承認が必要な場合は承認段を分ける前提とする

## Consequences

### Positive

- 状態遷移ロジックが単純になる
- 実装とテストの難易度を抑えられる

### Negative

- 全員承認などのユースケースには対応できない

## Alternatives Considered

### 1. 全員承認必須

- 不採用理由
  - 状態管理が複雑になる

### 2. 主担当 + 副担当

- 不採用理由
  - Phase1の範囲を超える
